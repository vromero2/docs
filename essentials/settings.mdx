## Configuración de la Clínica

### Datos Básicos

**Información General**:
- Nombre completo de la clínica
- Dirección física completa
- Teléfonos de contacto
- Email principal
- Sitio web (opcional)
- Logo de la clínica

**Horarios de Atención**:
- Días de la semana que atiendes
- Horarios de apertura y cierre
- Días festivos y vacaciones
- Horarios especiales por día

### Configuración Financiera

**Moneda y Formato**:
- Moneda local (ARS, USD, EUR, etc.)
- Formato de números decimales
- Símbolo de moneda ($, €, etc.)
- Separadores de miles y decimales

**Configuración de Pagos**:
- Métodos de pago aceptados
- Porcentaje de descuento por pago contado
- Días de gracia para pagos
- Intereses por mora

## Gestión de Usuarios

### Roles del Sistema

**Doctor**:
- Acceso completo a expedientes
- Creación y edición de presupuestos
- Acceso al odontograma
- Gestión de tratamientos

**Secretaria**:
- Gestión de agenda
- Creación de presupuestos
- Registro de pagos
- Gestión básica de pacientes

**Auxiliar**:
- Carga de multimedia
- Lectura de expedientes
- Soporte en consultas

**Administrador**:
- Acceso completo al sistema
- Gestión de usuarios
- Configuración del sistema
- Reportes avanzados

### Permisos Específicos

| Permiso | Descripción | Roles |
|---------|-------------|-------|
| `view_patients` | Ver lista de pacientes | Todos |
| `create_patient` | Crear nuevos pacientes | Doctor, Secretaria, Admin |
| `edit_patient` | Editar datos de pacientes | Doctor, Admin |
| `delete_patient` | Eliminar pacientes | Solo Admin |
| `view_appointments` | Ver agenda | Doctor, Secretaria, Admin |
| `create_appointment` | Crear citas | Doctor, Secretaria, Admin |
| `edit_appointment` | Modificar citas | Doctor, Secretaria, Admin |
| `view_budgets` | Ver presupuestos | Doctor, Secretaria, Admin |
| `create_budget` | Crear presupuestos | Doctor, Secretaria, Admin |
| `approve_budget` | Aprobar presupuestos | Doctor, Admin |
| `view_payments` | Ver pagos | Secretaria, Admin |
| `create_payment` | Registrar pagos | Secretaria, Admin |
| `view_reports` | Ver reportes | Admin |
| `manage_users` | Gestionar usuarios | Solo Admin |
