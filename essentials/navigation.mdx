## Estructura Principal del Sistema

### Barra de Navegación Superior

**Logo y Nombre**: 
- Clic en el logo para volver al dashboard principal
- Muestra el nombre de tu clínica configurado

**Menú de Usuario**:
- Foto de perfil
- Nombre del usuario actual
- Opciones: Perfil, Configuración, Cerrar Sesión

**Notificaciones**:
- Icono de campana con contador
- Alertas de citas próximas
- Notificaciones del sistema
- Mensajes de otros usuarios

### Menú Lateral Principal

**Dashboard**:
- Vista general de la clínica
- Métricas principales
- Accesos rápidos
- Gráficos de actividad

**Pacientes**:
- Lista de todos los pacientes
- Búsqueda y filtros
- Crear nuevo paciente
- Importar desde Excel

**Agenda**:
- Vista diaria, semanal y mensual
- Crear nueva cita
- Gestión de horarios
- Recordatorios

**Presupuestos**:
- Lista de presupuestos
- Crear nuevo presupuesto
- Estados: Pendiente, Aprobado, Rechazado
- Odontograma integrado

**Tesorería**:
- Registro de pagos
- Gastos y egresos
- Reportes financieros
- Conciliación bancaria

**Administración**:
- Gestión de usuarios
- Configuración del sistema
- Categorías y tratamientos
- Laboratorios y proveedores

## Navegación por Módulos

### Módulo de Pacientes

**Vista de Lista**:
- Tabla con todos los pacientes
- Búsqueda en tiempo real
- Filtros por estado, fecha, etc.
- Ordenamiento por columnas

**Ficha del Paciente**:
- Pestañas: Datos, Expediente, Multimedia, Presupuestos
- Navegación rápida entre secciones
- Historial de cambios
- Acciones contextuales

**Odontograma**:
- Vista interactiva de la boca
- Clic en dientes para editar
- Símbolos de diagnóstico y tratamiento
- Historial de cambios

### Módulo de Agenda

**Vista Diaria**:
- Horarios de 8:00 a 20:00
- Citas por consultorio
- Estados visuales (confirmada, pendiente, cancelada)
- Drag & drop para mover citas

**Vista Semanal**:
- 7 días en vista horizontal
- Resumen de citas por día
- Indicadores de ocupación
- Acceso rápido a cada día

**Vista Mensual**:
- Calendario completo del mes
- Indicadores de días con citas
- Navegación entre meses
- Vista de disponibilidad

### Módulo de Presupuestos

**Lista de Presupuestos**:
- Estados con colores distintivos
- Filtros por estado, fecha, paciente
- Búsqueda por número o paciente
- Acciones masivas

**Creación de Presupuesto**:
- Selección de paciente
- Odontograma para marcar tratamientos
- Cálculo automático de totales
- Opciones de financiamiento

**Aprobación**:
- Flujo de aprobación configurable
- Notificaciones automáticas
- Comentarios y observaciones
- Historial de cambios

## Atajos de Navegación

### Teclado

| Atajo | Acción | Módulo |
|-------|--------|--------|
| `Ctrl + 1` | Ir a Dashboard | Global |
| `Ctrl + 2` | Ir a Pacientes | Pacientes |
| `Ctrl + 3` | Ir a Agenda | Agenda |
| `Ctrl + 4` | Ir a Presupuestos | Presupuestos |
| `Ctrl + 5` | Ir a Tesorería | Tesorería |
| `Ctrl + 6` | Ir a Administración | Administración |
| `Ctrl + F` | Buscar | Todos |
| `Ctrl + N` | Nuevo elemento | Contextual |
| `Esc` | Cerrar modal/formulario | Global |

### Mouse

**Clic Izquierdo**:
- Seleccionar elementos
- Abrir formularios
- Navegar entre páginas

**Clic Derecho**:
- Menú contextual
- Acciones rápidas
- Opciones adicionales

**Doble Clic**:
- Editar elemento
- Abrir en nueva pestaña
- Acción principal

## Breadcrumbs y Ubicación

### Indicador de Ubicación

**Formato**: `Dashboard > Pacientes > Juan Pérez > Expediente`

**Funcionalidad**:
- Muestra la ruta actual
- Permite navegación hacia atrás
- Indica el contexto actual
- Facilita la orientación

### Navegación Contextual

**Botones de Acción**:
- Siempre visibles según el contexto
- Acciones más frecuentes
- Estados claros (habilitado/deshabilitado)
- Iconos descriptivos

**Enlaces Relacionados**:
- Sugerencias de navegación
- Accesos rápidos
- Información relacionada
- Próximos pasos sugeridos

## Responsive Design

### Navegación Móvil

**Menú Hamburger**:
- Icono de tres líneas
- Menú desplegable completo
- Navegación por pestañas
- Búsqueda integrada

**Gestos Táctiles**:
- Swipe para cambiar vistas
- Pinch para zoom
- Tap para seleccionar
- Long press para opciones

### Navegación Tablet

**Vista Adaptativa**:
- Menú lateral colapsable
- Contenido principal expandido
- Botones de tamaño táctil
- Navegación optimizada

## Personalización de la Navegación

### Configuración de Usuario

**Menú Personalizado**:
- Mostrar/ocultar módulos
- Reordenar elementos
- Accesos rápidos personalizados
- Favoritos

**Preferencias de Vista**:
- Densidad de información
- Tamaño de fuente
- Tema claro/oscuro
- Idioma

### Configuración Administrativa

**Permisos de Navegación**:
- Restringir acceso a módulos
- Ocultar funciones específicas
- Personalizar según roles
- Configuración por usuario

**Branding**:
- Logo personalizado
- Colores de la clínica
- Nombre del sistema
- Información de contacto

**Consejo**: Usa los atajos de teclado para navegar más rápido. La mayoría de las acciones se pueden realizar sin usar el mouse.
