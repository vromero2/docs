## Comienza en tres pasos

Configura tu clínica dental y comienza a gestionar pacientes, citas y tratamientos.

### Paso 1: Acceso al sistema

<AccordionGroup>
  <Accordion icon="user-check" title="Iniciar sesión en OCLÜ">
    Accede a tu cuenta de OCLÜ CRM en [www.oclucrm.com](https://www.oclucrm.com)
    
    - Usa las credenciales proporcionadas por tu administrador
    - Si es tu primera vez, sigue el proceso de configuración inicial
    - Verifica que tu rol tenga los permisos necesarios
  </Accordion>
  <Accordion icon="gear" title="Configuración inicial">
    1. Completa tu perfil de usuario
    2. Configura los datos de tu clínica
    3. Define horarios de atención
    4. Establece categorías de tratamientos
    
    <Tip>La configuración inicial es clave para aprovechar todas las funcionalidades.</Tip>
  </Accordion>
</AccordionGroup>

### Paso 2: Primeros pasos según tu rol

<AccordionGroup>
  <Accordion icon="user-doctor" title="Si eres Doctor">
    Comienza creando tu primer paciente y explorando el odontograma:
    
    1. Ve a **Pacientes** → **Agregar Paciente**
    2. Completa los datos básicos
    3. Accede al **Odontograma** para registrar el estado dental
    4. Explora la **Agenda** para programar citas
    
    <Tip>El odontograma es tu herramienta principal para el diagnóstico.</Tip>
  </Accordion>
  <Accordion icon="user-tie" title="Si eres Secretaria">
    Enfócate en la gestión administrativa:
    
    1. Configura la **Agenda** con horarios disponibles
    2. Crea **Presupuestos** para tratamientos
    3. Gestiona **Pagos** y cobros
    4. Importa pacientes desde Excel si tienes una base existente
    
    <Tip>La agenda es tu centro de control para la organización diaria.</Tip>
  </Accordion>
  <Accordion icon="shield-check" title="Si eres Administrador">
    Configura el sistema completo:
    
    1. Crea **Usuarios** y asigna roles
    2. Configura **Permisos** específicos
    3. Establece **Categorías** de tratamientos
    4. Configura **Laboratorios** y proveedores
    
    <Tip>La configuración inicial determina la eficiencia del sistema.</Tip>
  </Accordion>
</AccordionGroup>

### Paso 3: Flujo de trabajo básico

<Accordion icon="workflow" title="Proceso típico de atención">
  1. **Recepción**: Busca o crea el paciente
  2. **Consulta**: Registra síntomas en el expediente
  3. **Diagnóstico**: Usa el odontograma para marcar problemas
  4. **Presupuesto**: Crea presupuesto con tratamientos necesarios
  5. **Programación**: Agenda próxima cita si es necesario
  6. **Pago**: Registra el pago realizado
  7. **Seguimiento**: Actualiza el expediente con la evolución

<Tip>Este flujo se adapta a cualquier tipo de tratamiento dental.</Tip>
</Accordion>
